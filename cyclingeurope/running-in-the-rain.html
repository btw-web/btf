Running in the rain

A slightly longer run this morning… continued down the Thames path instead of doubling back via the canal. It added another mile or so to the run but did require a few stops (the bits in yellow) to catch my breath. The last stretch – all the way down the A4 from Cemetry Junction to the Royal Berks Hospital – was done in very, very wet conditions! I’ll aim to do the whole thing again next weekend without the yellow bits and perhaps even managing to start and finish in the same place!
