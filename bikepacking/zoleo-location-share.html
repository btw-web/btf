ZOLEO Introduces Unlimited Live Tracking

ZOLEO is a global messaging device that’s designed to work with your smartphone. The device uses satellite, cellular, or WiFi coverage to send and receive messages virtually, anywhere on the planet, and it has an SOS function for emergency scenarios. Until recently, ZOLEO’s main focus was on messaging, but it didn’t allow for live tracking, which is great for bikepacking trips when you’re covering a lot of ground each day.

Their new Location Share+ feature allows the user to share their location with up to five check-in contacts on a selected interval from every six minutes to four hours. Using the free ZOLEO app, the user and their contacts can view the user’s current location and route breadcrumbs, with or without cellular coverage.

“Location Share+ was built in response to market feedback,” explains Morris Shawn, President of ZOLEO Inc. and Roadpost Inc. “While all ZOLEO users can add their location coordinates to a message on-demand, some users also valued enhanced tracking that would let them share real-time updates with close contacts, and have their current location and full breadcrumb trail displayed in a map. Location Share+ was introduced as a low-cost option for those users, and it includes unlimited location sharing and check-in messages.”

ZOLEO Specs

  • Weight: 150g
  • Size: 3.58 x 2.6 x 1.06″
  • Protection: IP68, Shock-resistant: MIL-STD 810G
  • Power Input: Micro-USB Type B
  • Battery: Rechargeable internal Lithium Ion
  • Power: 200+ hours
  • Satellite Network: Iridium
  • Global Navigation Networks: GPS, GLONASS

The ZOLEO device retails for $199 USD and monthly plans cost between $20 and $50. The Location Share+ add-on costs an additional $6 per month and can be added or removed from a plan at any time.

Learn more at Zoleo.com.
