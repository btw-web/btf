2020 Norco Search XR Carbon Lineup

I reviewed the 2018 Norco Search XR Carbon, the most recent (and highest end) model within the lineup when the Search XR lineup was revamped. That particular release saw a redesigned frame with modern gravel bike features and upgrades, like larger tire clearance and lots of mounting options, as well as several build options in both carbon and steel.

Although not much has changed for 2020, Norco has introduced some great new paint jobs and updated build kits, and I think they’re looking pretty good. For those who can’t live without a dropper post, you’ll be happy to see the X-Fusion Manic Gravel 50mm Dropper post specced on all but the three base models, and it looks like the dropper remote has been configured into the brake lever, addressing an issue I had with the 2018 Search XR’s dropper lever placement. Check out the details and summarized component lists below.

2020 Norco Search XR C SE / XR C1

No longer is the SRAM Force 1 build the highest end Search XR model available. For 2020, Norco has introduced a SRAM Force eTap 12-Speed build (XR C SE) and a Shimano GRX 11-speed build (XR C1). Both models are specced with Easton EC70 AX Carbon handlebars, X-Fusion Manic Gravel 50mm Dropper posts, and 700×42 WTB Resolute tires. Both the XR C SE and XR C1 are available in a ‘Army Green/Warm Grey’ paint job.

Norco Search XR C SE Build

  • SRAM Force eTap AXS 12-Speed Drivetrain
  • SRAM Force Hydraulic Disc Brakes
  • Reynolds ATR X Carbon Rims
  • Reynolds Hubs
  • SRAM Force 1 DUB Carbon Crankset 42T
  • $5,999 USD

Norco Search XR C1 Build

  • Shimano GRX 11-Speed Drivetrain
  • Shimano GRX Hydraulic Disc Brakes
  • Reynolds ATR Carbon Rims
  • Reynolds Hubs
  • Easton EC90 Carbon Crankset 40T
  • $4,999 USD

2020 Norco Search XR C2 / C3

Both the Search XR C2 and C3 models will have a price tag under $4,000, but still offer the comfort of Norco’s high modulus carbon frame and fork. The C2 sees the same Shimano GRX drivetrain as the higher end XR C1 build above, but an SLX level cassette instead of XT, fewer carbon components, and Easton EA70 AX Wheels. The XR C3 switches things up with a 2×11 Shimano GRX drivetrain and more affordable WTB ST Light wheels. The Norco Search XR C2 comes in ‘Metallic Brown Fade’ and the C3 is available in ‘Blue Fade’ for 2020.

Norco Search XR C2 Build

  • Shimano GRX 11-Speed Drivetrain
  • Shimano GRX BR-RX810 Hydraulic Disc Brakes
  • Easton EA70 AX Wheels
  • Easton EA90 Crankset 40T
  • $3,799 USD

Norco Search XR C3 Build

  • Shimano GRX 2×11 Drivetrain
  • Shimano GRX BR-RX400 Hydraulic Disc Brakes
  • WTB ST Light i23 Wheels
  • Shimano GRX Crankset 46/30T
  • $3,299 USD

The more affordable, yet still totally capable, steel and aluminum Search XR models have also received fresh paint and new build kits, ranging in price from $1,499 to $2,699. Head over to Norco.com to see complete specs and learn more.
